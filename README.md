# dotfile

我自己的一些配置

包含：zsh, alacritty, vim, neofetch, i3, polybar, dunst

# Zsh

支持记录历史命令、报错声音、自动建议、历史查询、语法高亮、补全建议等

不基于oh-my-zsh，让你的shell更快、更顺手！
